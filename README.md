# Ulster Leinster URC Preview

Independent GitHub Pages project for Ulster Rugby vs Leinster Rugby in Round 15 of the BKT United Rugby Championship on April 17, 2026.

Planned live URL:

https://zeshanalikhan.github.io/ulster-leinster-urc-preview/

## Project Notes

- Separate project folder. It does not reuse or merge previous project files.
- Includes an original preview, time/venue page, table context, live updates page, FAQ, and optimized graphics.
- Live updates read from `data/live-updates.json`, so updates can be added manually or from an authorized data source without scraping official live commentary.
- Research notes are stored in `docs/research-notes.md`.
